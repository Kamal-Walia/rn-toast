# rn-toast

Simple JS based RN toast that will work on both iOS and android

## Installation

```sh
npm install rn-toast
```

## Usage

```js
import RnToast from "rn-toast";

// ...
toastRef.show(msg, 1000);
<RnToast ref={(toast) => (toastRef = toast)} />
```
